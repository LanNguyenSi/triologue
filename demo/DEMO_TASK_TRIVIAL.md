2. Gregor: Referenzprojekt Köln aufbereiten → bis KW13
3. Angebot vorbereiten: REST API Integration mit d.3 → Scope definieren

## Warum diese Demo funktioniert:
- **Einfach:** Jeder versteht Meeting-Protokolle
- **Relevant:** Das ist publicplan-Alltag
- **Schnell:** Agent liefert in <30 Sekunden
- **Mehrwert sichtbar:** Aus Fließtext → strukturierte Action Items
- **Easter Egg:** Agent erkennt AI-Strategie als Opportunity → zeigt "Mitdenken"
