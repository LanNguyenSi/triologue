# 2.2 Task Assignment Push

**Priorität:** HOCH
**Aufwand:** 2 Tage
**Phase:** 2 — Workflow-Reife

## Ziel

Wenn einem Agent eine Task zugewiesen wird, bekommt er automatisch den Task Execution Context via SSE/WebSocket gepusht. Der Agent muss nichts aktiv abfragen.

## Hintergrund

Heute muss ein Mensch im Chat "@lava bitte bearbeite die Aufgabe" schreiben. Der Agent muss dann selbst die Task finden, Context laden, etc. Mit Push bekommt der Agent eine strukturierte Notification mit allem was er braucht.

## Anforderungen

### Event-Typ

Neues SSE Event: `task.assigned`

```json
{
  "type": "task.assigned",
  "taskId": "cmmn4atdg...",
  "projectId": "cmmmcwmie...",
  "roomId": "cmmmcwmie-room-...",
  "assignedBy": "lan",
  "context": { ... }          // Vollstaendiger Task Execution Context (siehe Task 1.1)
}
```

### Trigger

Event wird gesendet wenn:
1. `assignedTo` auf einer Task geaendert wird (PATCH)
2. Eine neue Task mit `assignedTo` erstellt wird (POST)
3. Nur wenn der Assignee ein Agent ist (userType = 'agent')

### SSE Gateway

Der Triologue Agent Gateway muss `task.assigned` Events an den richtigen Agent routen (basierend auf `assignedTo` User-ID → Agent SSE Connection).

### Agent-Verhalten

Der Agent entscheidet autonom ob er die Task annimmt:
- Wenn ja: Status auf `in_progress` setzen, mit der Arbeit beginnen
- Wenn nein: Nachricht im Room posten warum (z.B. "Task ausserhalb meines Kompetenzbereichs")

## Abhaengigkeiten

- **Task 1.1** (Task Runtime Context API) muss zuerst fertig sein — der Push-Payload nutzt denselben Context

## Dateien die geaendert werden muessen

1. **`server/src/routes/projects.ts`** — Bei Task-Create/Update: Event emittieren wenn assignedTo ein Agent ist
2. **`server/src/services/sseService.ts`** (oder aehnlich) — Event an Agent-SSE-Stream senden
3. **`triologue-agent-gateway/`** — task.assigned Event an OpenClaw Session weiterleiten
4. **`triologue-agent-gateway/examples/openclaw-sse-client.ts`** — task.assigned Handler

- [ ] Task-Zuweisung an menschlichen User erzeugt kein SSE Event
